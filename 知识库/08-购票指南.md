## 二、购票渠道

### 2.1 线上购票（推荐）

**微信小程序购票**：
1. 微信搜索「乐乐亲子乐园」小程序
2. 首页点击「在线购票」
3. 选择票种和数量
4. 选择游玩日期
5. 填写使用人信息
6. 确认订单并支付
7. 获得电子票二维码

**微信公众号购票**：
1. 关注「乐乐亲子乐园」公众号
2. 点击菜单「购票」→「在线购票」
3. 后续流程同小程序

**第三方平台**：
- 美团、大众点评、携程等平台均可购买
- 价格以平台显示为准，可能有平台专属优惠

### 2.2 现场购票

- **位置**：园区正门售票窗口
- **支付方式**：微信支付、支付宝、现金
- **营业时间**：08:30 起（比开园早 30 分钟）
- **注意**：现场购票按原价，不享受线上优惠

## 三、购票规则

### 3.1 购票须知

| 规则 | 说明 |
|------|------|
| **有效期** | 门票购买后限指定日期使用，过期作废 |
| **实名制** | 购票时需填写使用人姓名，入园时可能核验 |
| **限购** | 单笔订单最多购买 10 张票 |
| **儿童判定** | 以身高为主要判定标准，建议携带儿童以便测量 |
| **免票儿童** | 身高 < 0.8m 或年龄 < 1 周岁的儿童免票，需有成人陪同入园 |
| **长者票** | 60 岁以上老人凭身份证或老年证购买 |

### 3.2 入园须知

| 事项 | 说明 |
|------|------|
| **入园方式** | 出示电子票二维码，工作人员扫码验证后入园 |
| **入园时间** | 最晚入园时间为闭园前 1 小时 |
| **重复入园** | 门票为单次入园有效，出园后需重新购票 |
| **临时外出** | 如需临时外出（15 分钟内），可到信息服务中心登记 |
| **携带物品** | 允许自带食物和水入园（禁止酒精饮料、玻璃瓶、明火加热食品） |

## 四、退票与退款政策

### 4.1 退票规则

| 退票时间 | 退款比例 | 说明 |
|:--------:|:--------:|------|
| 游玩日前 3 天及以上 | 100% 全额退款 | 无条件退款 |
| 游玩日前 1-2 天 | 80% 退款 | 扣除 20% 手续费 |
| 游玩日当天（未入园） | 50% 退款 | 扣除 50% 手续费 |
| 已入园 | 不可退款 | — |
| 因天气闭园 | 100% 全额退款 | 园区发布闭园通知后自动退款 |

### 4.2 退票流程

1. 进入小程序「我的订单」
2. 选择需要退票的订单
3. 点击「申请退款」
4. 选择退款原因
5. 提交申请
6. 退款将在 1-3 个工作日内原路返回

### 4.3 特殊情况

| 情况 | 处理方式 |
|------|----------|
| **恶劣天气闭园** | 自动全额退款，已使用快速通道票一并退还 |
| **园区设备故障** | 对应项目快速通道票可退，普通票不退 |
| **订单重复支付** | 联系客服，核实后退还重复部分 |
| **第三方平台购票** | 请联系原购票平台处理退款 |

## 五、电子票使用说明

### 5.1 电子票内容

电子票包含以下信息：
- 订单编号
- 票种和数量
- 游玩日期
- 二维码（入园验证用）
- 使用人信息
- 订单金额

### 5.2 使用流程

```
到达园区正门
    │
    ▼
打开小程序「我的订单」
    │
    ▼
出示电子票二维码
    │
    ▼
工作人员扫码验证
    │
    ▼
验证通过 → 入园
```

### 5.3 常见问题

**Q：电子票二维码找不到怎么办？**
A：打开小程序 → 我的 → 我的订单 → 找到对应订单即可查看。

**Q：手机没电了怎么入园？**
A：可到正门信息服务中心，凭购票时使用的手机号查询订单并办理入园。

**Q：可以截图二维码入园吗？**
A：可以。但建议使用小程序直接展示，截图可能因过期而失效。

**Q：一张订单多张票，需要一起入园吗？**
A：不需要。每张票有独立二维码，可分开使用。

## 六、快速通道票

### 支持快速通道的项目

| 项目 | 快速通道价格 | 预计节省时间 |
|:----:|:-----------:|:-----------:|
| 🎢 过山车·极速飞鹰 | ¥30/人/次 | 约 30-45 分钟 |
| 🎡 摩天轮·梦幻之眼 | ¥20/人/次 | 约 20-30 分钟 |

### 购买方式

- 在对应项目入口处扫码购买
- 或在小程序「项目详情」页购买
- 会员可用 200 积分兑换 1 次快速通道

### 使用规则

- 快速通道票限购买当日使用
- 每人每次限购买 1 张
- 凭快速通道二维码走专用通道
- 因天气原因项目关闭可退票，个人原因不退
