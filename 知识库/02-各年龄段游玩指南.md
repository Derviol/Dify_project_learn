# 👶 各年龄段游玩指南

> 本文档按儿童发育阶段提供游玩建议、项目匹配和安全注意事项。
> 绑定 Agent：🎯 子 Agent 1「乐乐·游玩规划师」

---

## 总览：年龄分段与游玩策略

| 年龄段 | 发育特点 | 游玩策略 | 推荐项目数/天 | 关键注意事项 |
|--------|----------|----------|:------------:|-------------|
| 🍼 **0–3 岁** 婴幼儿 | 感官敏感期，体力有限，需要频繁休息和哺乳/换尿布 | 温和体验为主，单项目时长短，动静结合 | ≤ 4 个项目（含午休） | 婴儿车、母婴室、午休 |
| 🧒 **3–6 岁** 学龄前儿童 | 好奇心强，喜欢互动，体能逐步发展 | 互动探索为主，适度挑战，兴趣引导 | 4–6 个项目 | 身高限制、防晒防暑 |
| 👦 **6–12 岁** 学龄儿童 | 体能充沛，追求挑战，有自主偏好 | 冒险刺激 + 科普深度体验并行 | 5–7 个项目 | 安全规范、同伴互动 |

## 🍼 0–3 岁婴幼儿游玩指南

### 发育特征

| 维度 | 说明 |
|------|------|
| **感知能力** | 视觉、听觉、触觉快速发展，对鲜艳色彩和柔和声音敏感 |
| **运动能力** | 0–1 岁：躺着/抱着/爬行；1–2 岁：学步、小跑；2–3 岁：跑跳、上下台阶 |
| **注意力** | 单次专注约 5–15 分钟，需要频繁变换刺激 |
| **体力** | 每 1–1.5 小时需要休息/进食/小睡 |
| **社交** | 对陌生环境和声音可能产生焦虑，需要安全感 |

### 推荐项目（按优先顺序）

| 优先级 | 项目 | 推荐理由 | 建议时长 |
|:------:|------|----------|:--------:|
| ⭐⭐⭐ | 🎠 旋转木马 | 节奏缓慢、色彩丰富、家长同坐 | 10 分钟 |
| ⭐⭐⭐ | 🚂 小火车 | 坐着观赏沿途风景，轻松不费力 | 15 分钟 |
| ⭐⭐⭐ | 🫧 泡泡海洋馆（室内） | 恒温 32°C，婴幼儿专属戏水区 | 20–30 分钟 |
| ⭐⭐⭐ | 🐠 海底隧道（室内） | 推车可通行，视觉刺激丰富 | 15–20 分钟 |
| ⭐⭐ | 🐦 鸟语林 | 有遮阳棚，可看鸟类和孔雀 | 20 分钟 |
| ⭐⭐ | 🐑 动物互动区 | 触摸温顺小动物（兔子/小羊） | 15–20 分钟 |

### 一日游推荐路线（0–3 岁）

```
09:00  入园（建议东门进）
09:15  🚂 小火车（15分钟）— 轻松开场熟悉园区
09:45  🐑 动物互动区（20分钟）— 摸摸小动物
10:30  🫧 泡泡海洋馆（30分钟）— 室内恒温玩泡泡
11:15  海洋主题餐厅午餐
12:00  🍼 午休（母婴室/休息区休息1-1.5小时）

13:30  🎠 旋转木马（10分钟）— 温和游乐
14:00  🐠 海底隧道（20分钟）— 推车看鱼
14:30  自由活动 / 提前离园（避免宝宝过度疲劳）

💡 重点：午休时间不可压缩，0-3岁宝宝的体力和情绪高度依赖充分休息
```

### ⚠️ 安全与注意事项

| 事项 | 说明 |
|------|------|
| 🛑 **水上项目** | 须全程穿戴防水纸尿裤 + 救生衣，单次玩水不超过 15 分钟 |
| 🛑 **动物接触** | 接触动物须家长手扶，仅限于温顺小型动物，接触后立即洗手 |
| 🧴 **防晒防暑** | 使用 SPF50+ 物理防晒霜，佩戴遮阳帽，每 30 分钟补水 |
| 🛒 **婴儿车** | 所有推车友好项目已标注，建议带轻便伞车 |
| 🍼 **哺乳/喂食** | 园区有 5 个母婴室，配备哺乳隔间、热水和微波炉 |
| 👕 **备用衣物** | 建议多带一套备用衣物（泡泡海洋馆可能弄湿） |

## 🧒 3–6 岁学龄前儿童游玩指南

### 发育特征

| 维度 | 说明 |
|------|------|
| **感知能力** | 语言表达能力增强，能描述喜好和感受 |
| **运动能力** | 跑跳自如，平衡感提升，可参与轻度体力项目 |
| **注意力** | 单次专注约 15–25 分钟，对互动性内容兴趣高 |
| **体力** | 可连续游玩 2–3 小时，但需要适当休息 |
| **社交** | 喜欢模仿和角色扮演，对表演和互动兴趣浓厚 |

### 推荐项目（按兴趣分类）

#### 🦁 喜欢动物

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🐑 动物互动区 | 喂小羊、摸兔子、骑小马 | 30–40 分钟 |
| 🐬 海洋动物表演 | 海豚海狮表演，互动环节 | 30 分钟 |
| 🐠 海底隧道 | 看鲨鱼海龟，边走边学 | 15–20 分钟 |
| 🐦 鸟语林 | 喂鹦鹉，看孔雀开屏 | 20–30 分钟 |

#### 🎭 喜欢表演

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🤡 小丑互动秀 | 参与互动游戏赢奖品 | 20 分钟 |
| 🎩 魔术秀 | 大型魔术表演（陪同观看） | 25 分钟 |
| 🎉 花车巡游 | 卡通人偶互动 | 25 分钟 |

#### 🚗 喜欢冒险

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🚗 碰碰车乐园 | 亲子碰碰车，家长陪同 | 8 分钟 |
| 🦕 恐龙科普馆 | 仿真恐龙 + 4D 电影 | 30 分钟 |
| 🎡 摩天轮·梦幻之眼 | 高空观景，轿厢有空调 | 20 分钟 |

### 一日游推荐路线（3–6 岁）

```
09:00  入园
09:15  🎠 旋转木马（10分钟）— 轻松开场
09:40  🐑 动物互动区（30分钟）— 与动物亲密接触
10:30  🐬 海洋动物表演（30分钟）— 建议提前15分钟入场
11:15  🐠 海底隧道（15分钟）— 推车看鱼
11:45  🦕 恐龙科普馆（30分钟）— 4D电影体验
12:30  主题餐厅午餐

13:30  🎡 摩天轮（20分钟）— 高空俯瞰全园
14:00  🤡 小丑互动秀（20分钟）— 参与互动游戏
14:30  🚂 小火车（15分钟）— 轻松游览园区
15:00  🚗 碰碰车乐园（8分钟）— 亲子碰碰车
15:30  自由活动/甜品站休息
16:30  🎉 花车巡游（25分钟）— 沿途观看
17:15  纪念品商店 / 离园
```

### ⚠️ 安全与注意事项

| 事项 | 说明 |
|------|------|
| 📏 **身高检查** | 碰碰车需 0.9m 以上，水上乐园需 0.8m 以上 |
| 👨‍👩‍👦 **家长陪同** | 碰碰车 3–6 岁需家长陪同驾驶；魔术秀含惊吓元素建议陪同 |
| 🌞 **防晒** | 户外活动较多，每 2 小时补涂防晒霜 |
| 💧 **补水** | 随身携带水壶，园区设有免费饮水点 |
| 🧼 **卫生** | 动物互动后立即洗手消毒 |

## 👦 6–12 岁学龄儿童游玩指南

### 发育特征

| 维度 | 说明 |
|------|------|
| **感知能力** | 逻辑思维发展，对科普知识兴趣浓厚 |
| **运动能力** | 体能充沛，可参与中高强度体力项目 |
| **注意力** | 单次专注约 30–45 分钟，可理解复杂规则 |
| **社交** | 喜欢同伴互动和竞争性活动 |
| **偏好** | 追求刺激感、成就感，有明显个人兴趣倾向 |

### 推荐项目（按兴趣分类）

#### 🎢 喜欢刺激

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🎢 过山车·极速飞鹰 | 80km/h，失重俯冲 | 3 分钟 |
| 🚗 碰碰车乐园 | 可独立驾驶，体验碰撞乐趣 | 8 分钟 |
| 🌊 水上乐园·浪花谷 | 滑梯 + 造浪池（当前维护中） | 60 分钟 |
| 🎡 摩天轮·梦幻之眼 | 高空观景 | 20 分钟 |

#### 🔬 喜欢科普

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🦕 恐龙科普馆 | 4D 电影 + 化石挖掘 | 30 分钟 |
| 🐠 海底隧道 | 海洋生物知识学习 | 20 分钟 |
| 🐦 鸟语林 | 鸟类品种识别，喂鹦鹉体验 | 30 分钟 |
| 🫧 泡泡海洋馆 | 适合陪弟弟妹妹同玩 | 20 分钟 |

#### 🎭 喜欢表演

| 项目 | 理由 | 建议时长 |
|------|------|:--------:|
| 🎩 魔术秀 | 专业舞台魔术效果震撼 | 25 分钟 |
| 🎉 花车巡游 | 氛围热闹 | 25 分钟 |
| 🐬 海洋动物表演 | 了解动物行为训练 | 30 分钟 |

### 一日游推荐路线（6–12 岁）

```
08:45  入园（建议西门进，人较少）
09:00  🎢 过山车·极速飞鹰（3分钟+排队）
        → 使用快速通道，节省排队时间
09:30  🚗 碰碰车乐园（8分钟）
10:00  🦕 恐龙科普馆（30分钟）
        → 4D电影建议10:30场次
10:45  🐠 海底隧道（20分钟）
11:15  🐬 海洋动物表演（30分钟）— 14:00场次需等，改上午场
12:00  主题餐厅午餐

13:00  🎡 摩天轮·梦幻之眼（20分钟）— 最高点看园区
13:30  🐦 鸟语林（30分钟）— 喂鹦鹉
14:30  🎩 魔术秀（25分钟）— 梦幻剧场
15:30  挑战类项目二刷 + 自由活动
16:30  🎉 花车巡游（25分钟）— 中央广场最佳观赏位
17:15  纪念品商店 / 离园
```

### ⚠️ 安全与注意事项

| 事项 | 说明 |
|------|------|
| 📏 **身高严格限制** | 过山车需 ≥ 1.2m，碰碰车需 ≥ 0.9m，严格测量 |
| 🚫 **健康禁忌** | 心脏病、高血压、癫痫、近期手术史禁止乘坐刺激项目 |
| 🎒 **自行管理** | 6 岁以上可尝试自己背小包，培养独立性 |
| 🧴 **防晒** | 户外活动多，每 2 小时补涂防晒霜 |
| 👫 **同伴互动** | 如有多个孩子，可考虑分组体验不同项目提高效率 |

## 多年龄段家庭游玩建议

### 两个孩子年龄不同怎么办？

| 情况 | 策略 | 建议路线 |
|------|------|----------|
| 一个 0–3 岁 + 一个 3–6 岁 | 以低龄宝宝的节奏为主，穿插大龄可独立体验时段 | 上午：共同项目（旋转木马/海底隧道/动物互动）；下午：轮流陪护，大龄去冒险区 |
| 一个 3–6 岁 + 一个 6–12 岁 | 大部分项目可共同体验，个别项目按身高分 | 共同项目占 70%，分头行动占 30% |
| 三个以上不同年龄段 | 建议"分小组+汇合"策略，利用表演和用餐时间汇合 | 入园时确定汇合点（中央广场/主题餐厅） |
