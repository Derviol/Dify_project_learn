# ⚠️ 安全游玩规则

> 本文档按项目类型和年龄段提供安全提示和游玩规则。
> 绑定 Agent：🎯 子 Agent 2「乐乐·园区百事通」

---

## 一、通用安全规则（全园区）

这些规则适用于所有入园游客，不分年龄和项目类型。

### 基本安全守则

| 序号 | 规则 | 说明 |
|:----:|------|------|
| 🅰️ | **家长全程看护** | 儿童在园区内任何区域均须有成人陪同，不要让孩子离开视线 |
| 🅱️ | **身高体重测量** | 体验受限项目前，务必在入口处测量身高，如不符合要求请勿强行体验 |
| 🅲 | **遵守排队秩序** | 排队时家长看管好孩子，不要插队、翻越护栏 |
| 🅳 | **听从工作人员指挥** | 工作人员的安全指示具有最高优先级，必须遵守 |
| 🅴 | **身体不适及时就医** | 如有头晕、恶心、胸闷等不适，立即停止游玩并前往医务室 |
| 🅵 | **紧急情况处理** | 保持冷静，按照工作人员指引疏散，医务室电话可在信息台查询 |
| 🅶 | **孕妇/特殊人群** | 孕妇、心脏病患者、近期手术者请勿乘坐刺激性设施 |

### 必备安全装备建议

| 装备 | 建议 |
|------|------|
| 👟 **鞋子** | 穿着舒适防滑的运动鞋，避免拖鞋、凉鞋 |
| 🧴 **防晒** | SPF50+ 防晒霜，户外活动每 2 小时补涂 |
| 🧢 **防晒帽** | 宽檐遮阳帽，保护宝宝面部和颈部 |
| 💧 **水壶** | 随身携带水壶，园区设有免费饮水点 |
| 👕 **备用衣物** | 多带一套衣物和毛巾（水上项目、出汗、弄湿备用） |
| 🤱 **纸尿裤** | 婴幼儿多备纸尿裤和湿巾 |

## 二、按项目类型的安全规则

### 🎢 游乐设施（过山车/碰碰车/旋转木马/摩天轮等）

#### 身高限制速查表

| 项目 | 身高限制 | 其他限制 |
|------|:--------:|----------|
| 🎢 过山车·极速飞鹰 | ≥ 1.2m | 心脏病/高血压/癫痫患者禁坐 |
| 🚗 碰碰车乐园 | ≥ 0.9m | 3–6 岁需家长陪同驾驶 |
| 🎠 旋转木马 | 无限制 | 婴幼儿需家长陪同 |
| 🚂 小火车 | 无限制 | 无特殊限制 |
| 🎡 摩天轮·梦幻之眼 | 无限制 | 恐高症者慎坐 |
| 🌊 水上乐园·浪花谷 | ≥ 0.8m | 需穿泳衣+救生衣 |

#### 游乐设施安全细则

**过山车·极速飞鹰**
- 🔴 **身高不足 1.2m 严禁乘坐**
- 🔴 心脏病、高血压、癫痫患者、孕妇禁止乘坐
- 🔴 近期做过手术或有颈腰椎疾病者慎坐
- 🟡 乘坐前取下眼镜、手机、硬币等松散物品
- 🟡 全程保持坐姿，双手抓紧扶手，不要站立或探出身体
- 🟡 如果感到严重不适，向工作人员示意停车（紧急制动按钮在座椅侧面）

**碰碰车乐园**
- 🟡 3–6 岁儿童须由家长陪同驾驶
- 🟡 6 岁以上儿童可独立驾驶，但需遵守场地规则
- 🟡 禁止故意高速正面碰撞、追撞停靠车辆
- 🟡 长发建议扎起，避免被卷入

**摩天轮·梦幻之眼**
- 🟡 恐高症者建议评估后再乘坐
- 🟡 轿厢内禁止站立、跳跃、晃动
- 🟡 每个轿厢限坐 4 人，请勿超载
- 🟡 轿厢门关闭后请勿自行打开

### 🌊 水上项目（泡泡海洋馆 / 水上乐园）

#### 按年龄段的安全规则

**0–3 岁婴幼儿**
| 规则 | 说明 |
|------|------|
| 🩱 **穿戴防水纸尿裤** | 必须穿戴专用防水纸尿裤，防止排泄物污染水质 |
| 🦺 **穿戴救生衣** | 全程穿戴园区提供的婴幼儿专用救生衣 |
| 🤲 **家长水中手扶** | 家长必须下水全程手扶，不可仅在岸边观看 |
| ⏱️ **单次不超过 15 分钟** | 婴幼儿体温调节能力弱，单次玩水 ≤ 15 分钟 |
| 🧣 **出水立即擦干包裹** | 出水后立即用干毛巾擦干身体，穿好衣物保暖 |
| 🌡️ **水温要求** | 婴幼儿区水温应 ≥ 32°C |

**3–6 岁儿童**
| 规则 | 说明 |
|------|------|
| 🦺 **穿戴救生衣** | 必须穿戴救生衣，不得擅自取下 |
| 👨‍👩‍👦 **家长陪同** | 须家长全程陪同，不得单独下水 |
| 🚫 **不得进入深水区** | 不得进入水深 > 50cm 区域 |
| ⏱️ **每 30 分钟休息一次** | 玩水每 30 分钟上岸休息 10 分钟 |
| 💧 **及时补水** | 玩水时容易忽略饮水，提醒孩子喝水 |

**6–12 岁儿童**
| 规则 | 说明 |
|------|------|
| 🦺 **建议穿戴救生衣** | 不会游泳者必须穿戴 |
| 👫 **结伴而行** | 建议结伴游玩，不要单独下水 |
| 🚫 **禁止跳水/推人** | 禁止从岸边或滑梯上跳水、推别人入水 |
| 🏊 **量力而行** | 根据自身游泳能力选择合适的区域 |

### 🐑 动物互动项目（动物互动区 / 鸟语林）

#### 按年龄段的安全规则

**0–3 岁婴幼儿**
| 规则 | 说明 |
|------|------|
| 🤲 **家长全程手扶** | 接触动物时家长必须扶着宝宝的手 |
| 🐰 **仅限温顺小型动物** | 仅限于触摸兔子、小羊等温顺小型动物 |
| 🚫 **禁止喂食** | 0–3 岁婴幼儿禁止自行喂食动物（可由家长代喂展示） |
| 🧼 **接触后立即洗手** | 园区提供洗手池和免洗消毒液 |

**3–6 岁儿童**
| 规则 | 说明 |
|------|------|
| 👨‍👩‍👦 **家长陪同** | 在家长陪同下可喂食指定动物 |
| 🥄 **使用专用喂食工具** | 不得用手指直接喂食，使用园区提供的夹子或长勺 |
| 🐑 **指定喂食区域** | 只能在指定区域喂食，不得追喂或用食物引诱动物 |
| 🧼 **接触后立即洗手消毒** | 提供洗手消毒设施 |

**6–12 岁儿童**
| 规则 | 说明 |
|------|------|
| 👤 **可独立体验** | 可在家长视线范围内独立完成喂食体验 |
| 🌿 **尊重动物** | 不追赶、惊吓、拉扯动物 |
| 🧼 **接触后洗手** | 完成后必须洗手 |

### 🎭 表演类项目（海洋动物表演 / 魔术秀 / 小丑互动秀）

| 规则 | 说明 |
|------|------|
| 🪑 **提前入场就座** | 建议提前 15 分钟入场，避免开场后慌乱找座 |
| 🚫 **不要站立观看** | 演出期间请坐在座位上，不要站在过道上 |
| 📸 **关闭闪光灯** | 拍照时关闭闪光灯，避免影响演员和动物表演 |
| 🔊 **控制音量** | 家长提醒孩子不要大声喧哗，影响他人观演 |
| 👶 **安抚小月龄宝宝** | 如果宝宝哭闹，请暂时离场安抚后再返回 |
| 🎩 **魔术秀惊吓提醒** | 含炮声/突然出现道具等元素，胆小宝宝家长请评估 |

## 三、按年龄段的安全汇总

| 场景 | 0–3 岁 | 3–6 岁 | 6–12 岁 |
|------|:------:|:------:|:-------:|
| **玩水/水上项目** | 全程纸尿裤+救生衣，家长手扶，≤15 分钟 | 救生衣，家长陪同，避开深水区 | 建议救生衣，结伴而行 |
| **动物接触** | 仅摸温顺小动物，家长手扶 | 专用工具喂食，家长陪同 | 可独立体验，洗手即可 |
| **刺激项目** | ❌ 不推荐 | 仅限儿童版（如碰碰车低速档） | 身高达标可体验，注意健康限制 |
| **防晒** | SPF50+ 物理防晒 | SPF50+ 防晒霜+防晒帽 | SPF50+ 防晒霜 |
| **补水频率** | 每 30 分钟 | 每 30–45 分钟 | 每 45–60 分钟 |
| **午休必要性** | 必要（1–1.5 小时） | 建议（30–60 分钟） | 可选 |
| **防走失** | 婴儿车+防走失绳 | 防走失书包/通话手表 | 约定集合点和时间 |

## 四、紧急情况处理指南

### 🆘 常见紧急情况应对

| 情况 | 应对措施 |
|------|----------|
| **孩子走失** | 1️⃣ 原地等待 5 分钟（孩子可能自己返回）<br>2️⃣ 联系最近的工作人员或前往信息台<br>3️⃣ 园区广播寻人（提供孩子的姓名、衣着特征）<br>4️⃣ 报警求助 |
| **孩子受伤** | 1️⃣ 轻伤 → 就近找工作人员领取急救包或前往医务室<br>2️⃣ 重伤/昏迷 → 立即呼叫工作人员拨打 120<br>3️⃣ 医务室位置：梦幻城堡区旋转木马旁（近西门） |
| **突发疾病** | 1️⃣ 立即停止游玩，到阴凉/室内休息<br>2️⃣ 如有药物随身携带，立即服用<br>3️⃣ 情况严重立即到医务室就医或拨打 120 |
| **天气突变（雷暴）** | 1️⃣ 立即停止一切户外活动<br>2️⃣ 远离高大树木、金属栏杆、开阔地带<br>3️⃣ 就近进入室内场馆躲避（海洋探险区/科普区/梦幻剧场） |
| **中暑/热射病** | 1️⃣ 转移到阴凉通风处<br>2️⃣ 解开衣领，用湿毛巾冷敷额头和颈部<br>3️⃣ 少量多次补充淡盐水或运动饮料<br>4️⃣ 严重者立即就医 |
| **食物过敏** | 1️⃣ 立即停止进食可疑食物<br>2️⃣ 如已知过敏，使用随身携带的抗过敏药物<br>3️⃣ 严重过敏反应（呼吸困难/皮疹扩散）立即呼叫 120 |
