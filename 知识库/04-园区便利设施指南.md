## 二、🍼 母婴室详情

### 分布位置

| 编号 | 位置 | 距最近项目 | 设施 |
|:----:|------|:-----------:|:----:|
| 🍼 ① | **梦幻城堡区** | 旋转木马旁（步行约 3 分钟） | 哺乳隔间、换尿布台、热水、微波炉、婴儿床 |
| 🍼 ② | **海洋探险区** | 海洋餐厅旁（步行约 2 分钟） | 哺乳隔间、换尿布台、热水、微波炉 |
| 🍼 ③ | **欢乐农场区** | 动物互动区出口处 | 哺乳隔间、换尿布台、热水、婴儿秤 |
| 🍼 ④ | **科普区** | 恐龙科普馆入口旁 | 哺乳隔间、换尿布台、热水、温奶器 |
| 🍼 ⑤ | **冒险区** | 碰碰车乐园旁 | 哺乳隔间、换尿布台、热水 |

### 母婴室统一标准

| 项目 | 说明 |
|------|------|
| 🕐 **开放时间** | 09:00–18:00（与园区营业时间一致） |
| 🆓 **免费用品** | 所有母婴室均提供免费纸尿裤和湿巾，如有需要可向工作人员索取 |
| 💧 **热水供应** | 每个母婴室均配备热水壶和微波炉 |
| 🔒 **私密性** | 哺乳隔间有门帘/门锁，保障隐私 |
| 🧽 **消毒** | 每 2 小时清洁消毒一次，备有消毒湿巾和洗手液 |
| 👶 **婴儿床** | ①号母婴室配备婴儿小床，可供宝宝小睡 |

### 🍼 母婴室使用贴士

> 如果母婴室排队人多，以下区域也可哺乳：
> - 泡泡海洋馆休息区（室内恒温，舒适安静）
> - 海底隧道休息长椅（光线较暗，私密性好）
> - 各主题餐厅（可向服务员要一个安静角落的座位）
