# 🌤️ 天气应对指南

> 本文档按不同天气条件提供游玩建议、项目推荐和安全提示。

## 一、天气影响总览

| 天气 | 对游玩的影响 | 建议策略 |
|:----:|:-----------:|:---------:|
| ☀️ 晴好（15–30°C） | ✅ 最佳游玩天气 | 正常游玩，注意防晒 |
| ☀️ 高温（>35°C） | ⚠️ 中暑风险高，户外暴晒 | 室内项目为主，定时补水休息 |
| 🌧️ 雨天 | ⚠️ 户外项目可能调整 | 室内项目为主，备好雨具 |
| ⛈️ 雷暴 | 🔴 户外活动全部暂停 | 室内场馆躲避，关注天气变化 |
| ❄️ 寒冷（<10°C） | ⚠️ 户外体感冷 | 室内项目为主，注意保暖 |
| 🌫️ 雾霾（AQI >150） | ⚠️ 呼吸道健康风险 | 室内项目为主，戴口罩 |
| 🌬️ 大风（>6 级） | ⚠️ 高风险项目暂停 | 避开高空项目 |

## 二、☀️ 晴好天气（15–30°C）— 最佳游玩日

### 天气特征

这是最适合游园的理想天气，各项设施正常开放，户外活动舒适。

### 游玩建议

| 方面 | 建议 |
|:----:|:------|
| 🎢 **项目选择** | 所有项目均可正常游玩 |
| 🗺️ **路线** | 户外 + 室内灵活组合，充分利用好天气体验户外项目 |
| 👕 **穿着** | 短袖 + 薄外套（早晚温差大），舒适运动鞋 |
| 🧴 **防晒** | 涂抹 SPF30+ 防晒霜 |

### 推荐路线策略

```
上午（户外项目为主，天气凉爽）
  🎢 过山车 → 🚗 碰碰车 → 🐑 动物互动区 → 🚂 小火车

中午（适当安排室内避暑）
  🐠 海底隧道 → 🦕 恐龙科普馆（室内空调）

下午（户外+室内结合）
  🎡 摩天轮 → 🎩 魔术秀 → 🎉 花车巡游
```

## 三、☀️ 高温天气（>35°C）— 防暑优先

### 天气风险评估

| 风险 | 说明 |
|:----:|:------|
| 🔴 **中暑风险** | 高温 + 高湿度环境下儿童中暑风险显著增加 |
| 🔴 **晒伤风险** | UV 指数高，婴幼儿皮肤娇嫩易晒伤 |
| 🟡 **体力消耗** | 高温天气体能消耗加快，需增加休息频率 |

### 游玩建议

#### 推荐项目（优先室内/有遮阳）

| 优先级 | 项目 | 降温特点 |
|:------:|:----:|:---------:|
| ⭐⭐⭐ | 🫧 泡泡海洋馆 | 室内恒温 32°C，玩水降温 |
| ⭐⭐⭐ | 🐠 海底隧道 | 室内空调开放 |
| ⭐⭐⭐ | 🦕 恐龙科普馆 | 室内空调开放 |
| ⭐⭐⭐ | 🎩 魔术秀 | 室内空调（梦幻剧场） |
| ⭐⭐ | 🎡 摩天轮 | 轿厢有空调 |
| ⭐⭐ | 🎠 旋转木马 | 有遮阳顶棚 |
| ⭐ | 🚂 小火车 | 车厢有遮阳棚 |
| ❌ | 🎢 过山车 | 暴晒区域，体感温度高 |
| ❌ | 🐑 动物互动区 | 户外无遮阳，正午避免 |

#### 🛑 高温安全规则（来自安全规则 SR004）

| 规则 | 说明 |
|:----:|:------|
| ⏰ **避开 11:00–15:00 户外时段** | 这是紫外线最强、气温最高的时段，建议安排室内项目 |
| 💧 **每 30 分钟补水一次** | 随身携带水壶，利用园区免费饮水点 |
| 🧴 **涂抹 SPF50+ 防晒霜** | 出门前涂一次，户外每 2 小时补涂一次 |
| 🧢 **佩戴遮阳帽** | 宽檐帽比棒球帽防晒效果更好 |
| 👕 **穿着浅色透气衣物** | 浅色棉麻材质衣物散热好 |
| 🆘 **中暑征兆识别** | 头晕、脸红、大量出汗后停止出汗、恶心→立即到阴凉处休息并补水 |

### 🔥 高温天推荐路线

```
09:00  入园（趁凉快先玩户外）
09:15  🎢 过山车（早上排队人少＋不晒）
09:45  🚗 碰碰车（有遮阳棚）
10:15  🐑 动物互动区（上午动物活跃）
11:00  🎩 魔术秀（转入室内避暑）
11:45  午餐（城堡主题餐厅，有空调）

12:30–14:00 🔥 高温时段 → 全部室内
  🐠 海底隧道（空调）
  🦕 恐龙科普馆（4D电影）
  🫧 泡泡海洋馆（玩水降温）

14:00  🐬 海洋动物表演（有遮阳顶棚）
15:00  🎡 摩天轮（轿厢空调）
15:30  室内项目收尾/甜品站
16:30  🎉 花车巡游（傍晚没那么晒）
```

## 四、🌧️ 雨天游玩指南

### 天气风险评估

| 雨量 | 对游玩的影响 |
|:----:|:-------------|
| 🌤️ 小雨（<5mm/h） | 大部分户外项目正常开放，部分表演有遮雨设施 |
| 🌧️ 中雨（5–15mm/h） | 户外项目可能间歇性暂停，海洋动物表演可能调整 |
| ⛈️ 暴雨/雷暴（>15mm/h） | 户外活动全部暂停，请就近躲避 |

### 游玩建议

#### 推荐项目（室内避雨首选）

| 优先级 | 项目 | 避雨特点 |
|:------:|:----:|:---------:|
| ⭐⭐⭐ | 🫧 泡泡海洋馆 | 室内项目，不受天气影响 |
| ⭐⭐⭐ | 🐠 海底隧道 | 室内项目，不受天气影响 |
| ⭐⭐⭐ | 🦕 恐龙科普馆 | 室内项目，不受天气影响 |
| ⭐⭐⭐ | 🎩 魔术秀 | 室内剧场，不受天气影响 |
| ⭐⭐ | 🎠 旋转木马 | 有遮雨顶棚（小雨可用） |
| ⭐⭐ | 各主题餐厅 | 可坐下休息用餐避雨 |
| ❌ | 🎢 过山车 | 雨天湿滑，可能暂停 |
| ❌ | 🎉 花车巡游 | 雨天取消或简化 |
| ❌ | 🐬 海洋动物表演 | 视雨量决定是否调整 |

#### 🛑 雨天安全规则（来自安全规则 SR005）

| 规则 | 说明 |
|:----:|:------|
| 🚶 **远离高大树木和金属设施** | 雷雨天气存在雷击风险 |
| 🧊 **注意防滑** | 湿滑路面行走时放慢脚步，抱好小宝宝 |
| ☂️ **使用雨具** | 建议携带雨伞或雨衣（园区商店有售，¥15/件） |
| 🎪 **就近躲避** | 附近室内场馆均可躲避：海洋探险区、科普区、梦幻剧场、各餐厅 |
| 🔌 **避免接触电器设备** | 雨天远离充电桩等户外电器设施 |

### 🌧️ 雨天一日游推荐路线

```
09:00  入园（带好雨具）
09:15  🫧 泡泡海洋馆（室内恒温，玩泡泡）
10:00  🐠 海底隧道（室内看鱼，推车通行）
10:45  🦕 恐龙科普馆（4D电影+恐龙模型）
11:30  海洋主题餐厅午餐（就近用餐）

12:30  🎩 魔术秀（梦幻剧场室内）
13:30  🎠 旋转木马（有遮雨顶棚，小雨可玩）
14:00  逛纪念品商店
14:30  自由活动/甜品站

💡 如果雨势增大，建议：
- 延长在恐龙科普馆或泡泡海洋馆的停留时间
- 转移到商场式室内休息区
- 提前离园（注意停车场地面湿滑）
```

## 十、天气相关常见问题

### Q：高温天泡泡海洋馆需要预约吗？

**不需要预约**。泡泡海洋馆是室内恒温项目，先到先体验。如果排队人较多，可先取号等待，期间在海洋探险区室内逛逛。

### Q：下雨天门票可以退改吗？

小雨不影响正常运营，门票不可退改。如遇恶劣天气（台风/暴雨红色预警等），园区会发布闭园通知，届时可通过原购票渠道申请全额退款。

### Q：天气突然变化，已经买的快速通道票怎么办？

如因天气原因（雷暴/大风）导致快速通道对应项目关闭，可在项目入口处办理退票。因个人原因不体验，快速通道票不可退。
