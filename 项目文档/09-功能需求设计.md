## 2. 票务系统

### 2.1 用户故事

| 编号 | 角色 | 故事 | 优先级 |
|------|------|------|:------:|
| T-001 | 游客 | 我想在小程序上提前购买门票，避免现场排队 | P0 |
| T-002 | 游客 | 我想选择游玩日期和票种（成人/儿童/家庭套票） | P0 |
| T-003 | 游客 | 我想用微信支付快速完成付款 | P0 |
| T-004 | 游客 | 购票后我想获得电子票二维码，入园时扫码即可 | P0 |
| T-005 | 游客 | 我想查看历史订单和电子票 | P0 |
| T-006 | 游客 | 如果计划有变，我想申请退票退款 | P1 |
| T-007 | 游客 | 我想使用优惠码享受折扣 | P1 |

### 2.2 交互流程

```
在线购票流程：

选择票种 → 选择日期 → 填写游客信息 → 确认订单 → 微信支付 → 获得电子票
    │                                      │
    │          ┌── 使用优惠码？── 是 ──→ 输入码 → 验证 → 折扣
    │          └── 否 → 跳过
    │
    ▼
电子票页面：显示二维码 + 订单信息 + 入园须知

入园验票流程：
工作人员扫码 → 验证票有效性 → 更新票状态 → 闸机放行
```

## 3. AI 智能助手（增强）

### 3.1 相比原 Coze 版的增强点

| 增强点 | 说明 |
|--------|------|
| **语音输入** | 支持语音转文字输入，适合带娃不方便打字场景 |
| **卡片式回复** | 项目推荐、路线规划等展示为可交互卡片 |
| **购票咨询** | 新增票务相关的问答能力 |
| **上下文记忆增强** | 跨会话记住用户画像（孩子年龄、兴趣） |
| **主动推荐** | 基于位置和时间主动推送游玩建议 |
| **多模态** | 支持图片识别（拍照识别项目/设施）（P2） |

### 3.2 用户故事

| 编号 | 故事 | 对应 Agent |

| AI-002 | 我问"帮我规划一日游"，系统生成个性化路线 | 游玩规划师 |
| AI-003 | 我问"旋转木马排多久"，系统返回实时排队时间 | 园区百事通 |
| AI-004 | 我问"母婴室在哪"，系统返回最近母婴室位置 | 园区百事通 |

## 5. 会员系统

### 5.1 会员等级

| 等级 | 名称 | 条件 | 权益 |
|:----:|------|------|------|
| 1 | 普通会员 | 注册即得 | 基础服务 |
| 2 | 银卡会员 | 累计消费 ≥ 500元 或 游玩 ≥ 3次 | 9.5折购票 + 生日礼品 |
| 3 | 金卡会员 | 累计消费 ≥ 2000元 或 游玩 ≥ 10次 | 9折购票 + 快速通道1次/月 + 专属活动 |
| 4 | 钻石会员 | 累计消费 ≥ 5000元 或 游玩 ≥ 25次 | 8.5折购票 + 快速通道3次/月 + VIP休息室 |

### 5.2 积分规则

| 行为 | 积分 |
|------|:----:|
| 购票消费 1元 | +1 分 |
| 每日签到 | +5 分 |
| 发布游记（被审核通过） | +20 分 |
| 项目评价 | +10 分 |
| 邀请好友注册 | +50 分 |
| 兑换礼品 | 按礼品扣减 |
| 兑换快速通道 | -200 分 |

## 6. 游园打卡（P1）

### 6.1 打卡机制

| 机制 | 说明 |
|------|------|
| **项目打卡** | 完成项目体验后扫码打卡 |
| **集章系统** | 6大主题区各一枚印章，集齐有奖 |
| **成就徽章** | 完成特定目标解锁徽章（首次过山车、全家福打卡等） |
| **排行榜** | 月度打卡排行榜，前三名有奖 |
| **分享海报** | 生成精美打卡海报，一键分享朋友圈 |

## 7. 社区互动（P2）

### 8.1 内容类型

| 类型 | 说明 | 审核 |
|------|------|------|
| **游记** | 图文游记，分享游玩体验 | 人工审核 |
| **评价** | 对具体项目的评分和文字评价 | 自动审核（敏感词） |
| **攻略** | 游玩攻略、路线推荐 | 人工审核 |
| **提问** | 向其他游客提问 | 无需审核 |

### 8.2 互动功能

- 点赞、评论、收藏
- 关注其他用户
- 热门/最新/推荐排序
- 按标签筛选

## 8. 运营管理

### 9.1 数据看板指标

| 指标 | 展示方式 | 刷新频率 |
|------|---------|---------|
| 今日入园人数 | 数字 + 趋势 | 实时 |
| 实时在园人数 | 数字 | 5分钟 |
| 今日收入 | 数字 + 环比 | 实时 |
| 热门项目 TOP 10 | 柱状图 | 10分钟 |
| 排队热力图 | 热力图 | 5分钟 |
| 用户满意度 | 仪表盘 | 每日 |
| AI 对话量 | 折线图 | 每小时 |
| 意图分类分布 | 饼图 | 每日 |

### 9.2 内容管理

| 功能 | 说明 |
|------|------|
| **知识库管理** | 在线编辑知识文档，一键同步到 Dify |
| **项目信息编辑** | 修改项目描述、状态、图片等 |
| **活动管理** | 创建/编辑/发布活动，管理排期 |
| **公告发布** | 发布园区公告，推送到用户端 |
| **推送管理** | 管理各类推送通知模板 |
