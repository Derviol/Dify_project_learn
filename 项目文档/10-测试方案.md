## 2. 后端测试

### 2.1 单元测试示例

```python
# tests/test_ticket_service.py
import pytest
from app.services.ticket_service import TicketService

class TestTicketService:
    """票务服务单元测试"""

    @pytest.fixture
    def service(self, db_session):
        return TicketService(db_session)

    async def test_create_order_success(self, service, test_user, test_ticket_type):
        """创建订单成功"""
        order = await service.create_order(
            user_id=test_user.id,
            visit_date="2026-06-10",
            items=[{"ticket_type_id": test_ticket_type.id, "quantity": 2}],
        )
        assert order.order_no.startswith("T")
        assert order.total_amount == test_ticket_type.price * 2
        assert order.status == "pending"

    async def test_create_order_invalid_date(self, service, test_user):
        """创建订单-过去日期"""
        with pytest.raises(ValidationError, match="游玩日期不能早于今天"):
            await service.create_order(
                user_id=test_user.id,
                visit_date="2020-01-01",
                items=[{"ticket_type_id": 1, "quantity": 1}],
            )

    async def test_cancel_order(self, service, test_paid_order):
        """取消已支付订单"""
        result = await service.cancel_order(test_paid_order.order_no)
        assert result.status == "cancelled"
```

### 2.2 集成测试示例

```python
# tests/test_api_projects.py
import pytest
from httpx import AsyncClient

class TestProjectsAPI:
    """项目 API 集成测试"""

    async def test_list_projects(self, client: AsyncClient, init_data):
        """获取项目列表"""
        resp = await client.get("/api/v1/projects")
        assert resp.status_code == 200
        data = resp.json()
        assert data["code"] == 200
        assert len(data["data"]["items"]) == 15

    async def test_list_projects_filter_zone(self, client: AsyncClient, init_data):
        """按区域筛选项目"""
        resp = await client.get("/api/v1/projects?zone=梦幻城堡区")
        assert resp.status_code == 200
        items = resp.json()["data"]["items"]
        assert all(p["zone"] == "梦幻城堡区" for p in items)

    async def test_recommend_projects(self, client: AsyncClient, auth_headers, init_data):
        """智能推荐项目"""
        resp = await client.get(
            "/api/v1/projects/recommend?age=3&interests=动物",
            headers=auth_headers,
        )
        assert resp.status_code == 200
        items = resp.json()["data"]["recommendations"]
        assert len(items) > 0
        # 推荐结果应包含动物互动区
        names = [r["project"]["name"] for r in items]
        assert "动物互动区" in names
```

## 6. 安全测试

| 测试项 | 测试内容 | 工具 |
|--------|---------|------|
| **认证** | 未登录访问受保护接口返回 401 | pytest |
| **授权** | 普通用户访问管理接口返回 403 | pytest |
| **SQL 注入** | 输入 SQL 注入字符串，验证不生效 | pytest + sqlmap |
| **XSS** | 输入 script 标签，验证转义 | pytest |
| **CSRF** | 验证 CSRF Token 机制 | 手动测试 |
| **限流** | 超过限流阈值返回 429 | Locust |
| **JWT** | 过期 Token、篡改 Token | pytest |
| **儿童隐私** | 验证儿童信息加密存储 | 代码审查 |

## 7. CI/CD 集成

```yaml
# .github/workflows/test.yml
name: Test

on: [push, pull_request]

jobs:
  backend-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - name: Install dependencies
        run: pip install -r requirements.txt -r requirements-dev.txt
      - name: Run unit tests
        run: pytest tests/unit -v --cov=app --cov-report=xml
      - name: Run integration tests
        run: pytest tests/integration -v

  frontend-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: "20"
      - name: Install & Test
        run: |
          cd web
          npm install
          npm run test
          npm run lint
```
